# AI DDR Generator

An AI-powered system that converts inspection and thermal reports into a structured Detailed Diagnostic Report (DDR).

## Features
- PDF data extraction
- AI-based report generation
- Fallback system for reliability
- Clean structured output

## Tech Stack
- Python
- Flask
- Gemini AI
- PyMuPDF
- FPDF

## How to Run
1. Install requirements
2. Add API key in .env
3. Run app.py
4. Upload reports and generate DDR

## Key Highlight
Fault-tolerant system with fallback DDR generation even when API fails.
