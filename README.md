# AI DDR Generator

This project builds an AI system that converts inspection and thermal reports into a structured Detailed Diagnostic Report (DDR).

## Features
- Extracts data from PDFs
- Uses AI to merge and analyze reports
- Generates professional DDR output
- Handles missing/conflicting data

## Tech Stack
- Flask
- OpenAI API
- PyMuPDF
- FPDF

## How to Run
1. Install dependencies
2. Add OpenAI API key
3. Run app.py
4. Upload reports

Key Feature:
- Fault-tolerant system with fallback DDR generation
